# EHR Integration Platform

Welcome to the EHR Integration Platform documentation. This platform provides a comprehensive solution for integrating with Electronic Health Record (EHR) systems, specifically designed to work with Epic's FHIR API.

## Overview

The platform consists of multiple components:

- **Web Application**: Next.js frontend for user interaction
- **Server API**: Cloudflare Workers backend with tRPC
- **Documentation**: This Fumadocs-powered documentation site

## Key Features

- Patient Management (create, search, read, match)
- Appointment Scheduling (book, find, search, read)
- Billing and Administrative (accounts, coverage)
- Clinical Operations (notes and documents)

## Getting Started

1. Set up your Epic FHIR API credentials in the environment variables
2. Start the development server: `bun dev`
3. Access the web application at `http://localhost:3001`
4. View this documentation at `http://localhost:4000`
