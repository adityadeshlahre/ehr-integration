# Clinical Operations API

This section documents all clinical operations API endpoints.

## Get Binary Document

**Endpoint:** `http://localhost:3000/trpc/clinical.binary.readBinary`

**Method:** GET (query)

**Description:** Retrieves binary documents such as clinical notes, images, or other attachments.

**Input:** Binary document ID

## Search Binary Documents

**Endpoint:** `http://localhost:3000/trpc/clinical.binary.searchBinary`

**Method:** GET (query)

**Description:** Searches for binary documents.

**Input:** Search parameters for binary documents
